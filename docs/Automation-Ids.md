# CraftPlayer 自动化 ID

## Actions

- 播放音乐：`cn.craftine.craftplayer.play`
- 停止播放：`cn.craftine.craftplayer.stop`

## Rules

- 正在播放音乐：`cn.craftine.craftplayer.rule.is-playing`

## Triggers

- 音乐播放开始：`cn.craftine.craftplayer.trigger.session-started`
- 音乐播放结束：`cn.craftine.craftplayer.trigger.session-ended`
- 单曲播放结束：`cn.craftine.craftplayer.trigger.track-ended`
